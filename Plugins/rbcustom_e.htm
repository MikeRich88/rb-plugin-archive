About Micono RbIconUtil

  • 32 bit icon compatible.
  • Can add custom icon of image (for a full color) to files.
  • Read/Write an icon of various icon type of resource ID designated.
  • Ddelete an icon of various icon type of resource ID designated.

    * New method

    • mic_SetEachIcon (v1.0.0b11)

     

Methods

Micono CustomIcon (R1):

Add full-color custom icon to file.

mic_AddCustomIcon(f as FolderItem, g as Graphics) as integer
if the return value is not 0, this methods is in error.

Make Icons of resource ID designated.

mic_MakeIcons( f as FolderItem, g as Graphics, resID as Integer, iconType as Integer, dither as Boolean ) as Integer
if the return value is not 0, this methods is in error.
g: Image of icon to make it.
The another refer to instructions of R2.

Read an icon of various icon type of resource ID designated.

mic_GetEachIcon( f as FolderItem, resID as Integer, iconType as Integer, icns as Boolean ) as Picture
if the return value is not 0, this methods is in error.
The another refer to instructions of R2.

Write an icon of various icon type of resource ID designated.

mic_SetEachIcon( f as FolderItem, p as Picture, resID as Integer, iconType as Integer, icns as Integer ) as Integer
if the return value is not 0, this methods is in error.
Refer to instructions of R2.

Delete an icon of various icon type of resource ID designated.

mic_DeleteEachIcon( f as FolderItem, resID as Integer, iconType as Integer, resicns as Integer ) as Integer
if the return value is not 0, this methods is in error.
The another refer to instructions of R2.

Micono CustomIcon (R2):

Add full-color custom icon to file.

f.mic_AddCustomIcon(p as picture) as integer
if the return value is not 0, this methods is in error.
f: FolderItem

Make icons of resource ID designated.

f.mic_MakeIcons( p as picture, resID as Integer, iconType as Integer, dither as Boolean ) as Integer
if the return value is not 0, this methods is in error.
f: FolderItem
p: Image of icon to make it
resID: ID number of resource that icons are stored
iconType: Kind of icon made (Can several select it). See Table 1.
dither: Store dither image with icons less than 8bit

Read an icon of various icon type of resource ID designated.

f.mic_GetEachIcon( resID as Integer, iconType as Integer, icns as Boolean ) as Picture
f: FolderItem
resID: ID number of resource of icon read
iconType: Kind of icon made. See Table 1.
icns: Read an icon from 'icns' resource.

Write an icon of various icon type of resource ID designated.

f.mic_SetEachIcon( p as Picture, resID as Integer, iconType as Integer, icns as Integer ) as Integer
f: FolderItem
p: picture for Icon Image
resID: ID number of resource of icon read
iconType: Kind of icon made. See Table 1.
icns: 1=Resource only, 2='icns' only, 3=Resource + 'icns'

if the return value is not 0, this methods is in error.

Get picture from Resource data

mic_IconDataToPicture( iconData as String, iconType as Integer ) as Picture
iconData: String data by GetResource.
Exsample: data = app.ResourceFork.GetResource( "icns", 128 )
iconType: Kind of icon made. See Table 1.

Delete an icon of various icon type of resource ID designated.

f.mic_DeleteEachIcon( resID as Integer, iconType as Integer, resicns as Integer ) as Integer
if the return value is not 0, this methods is in error.
f: FolderItem
resID: ID number of resource of icon deleted
iconType: Kind of icon made. See Table 1.
resicns: 1=Resource only, 2='icns' only, 3=Resource + 'icns'

Get error code after working RbIconUtil method

mic_GetIconUtilError as integer
if the return value is not 0, this methods is in error.
