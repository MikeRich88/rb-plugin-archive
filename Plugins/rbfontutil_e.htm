About RBfontUtil

  • Get a font name from font ID.
  • Get font ID from a font name.
  • Examine whether there is a font of desired size.
  • Get font information.

Fix: mic_GetFontInfo method name

Methods

Get font name from font ID.

mic_GetFontName(fontNum as integer) as string

Å@

Get font ID from font name.

mic_GetFontNumber(fontName as string) as integer

Serch desiring font size.

mic_RealFont(fontNum as integer, fontSize as integer) as boolean

Get font information.

mic_GetFontInfo( infoType as integer ) as integer

infoType:
0: Height above base line
1: Depth below base line
2: Width of widest character in the font
3: Distance between lines

Exsample:

dim g as Graphics
dim ascent as integer

ascent = g.mic_GetFontInfo(0)
