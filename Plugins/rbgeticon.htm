Micono GetFinderIconについて

アイコン画像に関連した機能をもつREALbasic 2用のpluginです。
  • FolderItemからファインダで表示されているアイコンの画像を得ることができます。

　

変更点(->b9)

  • 指定したサイズのアイコンがない場合の不具合を修正（b9)
  • mig_GetFinderIconでHuge Iconを指定できるようにした（b9)
  • b7で、mic_GetFinderIconの32bit iconの8bit maskに不具合が出たのを修正(b8)
  • メソッドを追加（mic_GetFinderIconSuite）(b7)
  • IconServiceのないOSでの不具合を修正。（b6)
  • Plugin名を変更（旧：Micono GetIconPicture）。これに伴い、メソッド名も変更しました。※旧メソッド名も使えます。旧：Micono GetIconPictureを御利用の方はpluginsフォルダから削除して下さい。(b5)

　

  • Finder_icon Libを使用しています。（http://www.jwwalker.com/）

使い方

アイコン画像を得る。

f.mic_GetFinderIconSuite( Size as integer ) as Picture

戻り値：Picture
f: FolderItem

引き数

値

Size：アイコンサイズ

０ Large Icon (32*32)
１ Small Icon (16*16)
２ Mini Icon (16*12)
３ Huge Icon (48 * 48)

アイコン画像を得る。

f.mic_GetFinderIcon( Depth as integer, Size as integer, Transform as integer, BackColor as color ) as Picture

(旧メソッド：f.mic_GetIconPicture( Depth as integer, Size as integer, Transform as integer, BackColor as color ) as Picture)

戻り値：Picture
f: FolderItem

引き数

値

Depth：階調

32bit, 8bit, 4bit, 1bit、0 (Mask)

Size：アイコンサイズ

０ Large Icon (32*32)
１ Small Icon (16*16)
２ Mini Icon (16*12)
３ Huge Icon (48 * 48)

Transform：表示形式

&h0　何もしない
&h1　Disabled
&h2　Offline
&h3　Open
&h100　ラベル１
&h200　ラベル２
&h300　ラベル３
&h400　ラベル４
&h500　ラベル５
&h600　ラベル６
&h700　ラベル７
&h400　選択状態

BackColor：背景色

-

　

アイコンのマスク画像を得る。

f.mic_GetFinderIconMask( Size as integer ) as Picture

（旧メソッド：f.mic_GetIconMask( Size as integer ) as Picture）

戻り値：Picture
f: FolderItem

Size：アイコンサイズ　（上記を参照）
