About RBstringUtil

High-speed! (faster than CountFields, NthField, ReplaceAll of REALbasic)
Can select search method or ignore recognition of case of Enghlish characters !
Compatible with 1byte script and SJIS !
Compatible with strings containing "&h00" !
Can build Windows application !

Methods

 

Select search method of ignore or recognition case of Enghlish charactor.

mic_StrCheckType( CheckType as Integer )

CheckType

0: Ignore case (default: same search type as REALbasic).

1: Case sensitive (reset to 0 after work)

2: Case sensitive (fix case sensitive search)

Get result of RBstringUtil method

mic_StrError as Integer

Return value

0: No error.

1: Source is empty.

2: Separator is empty.

4: Wrong field number.

8: OldString is empty.

-108: Not enough memory in heap zone.

Returns the number of values (fields) in the string passed that are separated by the separator character passed.

mic_CountFields(source as String, separator as String) as Integer

Returns a field from a row of data.

mic_NthField(source as String, separator as String, fieldNumber as Integer) as String

Replaces the all occurrences of a string with another string.

mic_Replace(sourceString as String, oldString as String, newString as String) as String

Replaces the first occurrence of a string with another string.

mic_ReplaceAll(sourceString as String, oldString as String, newString as String) as String
