About Micono Mid

  • This plugin has functions same as Mid/MidB functions of REALbasic. The speed is almost equal with Mid/MidB functions of REALbasic.

    Compatible with 1byte script and SJIS !
    Compatible with strings containing "&h00" !
    Can build Windows application !

Methods
