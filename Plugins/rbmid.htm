Micono Midについて

  • REALbasicのMid関数、MidB関数と同様の機能を持ちます。速度はREALbasicのMid関数とさほど変わりません。

    ・SJIS対応！
    ・&h00を含む文字列対応！
    ・Windowsアプリケーション対応！

使い方

文字列の一部を返します。（文字数）

mic_Mid(Source as String, Start as Integer) as String

文字列の一部を返します(長さ指定）。（文字数）

mic_LenMid(Source as string, Start as Integer, Length as Integer) as String

文字列の一部を返します。（バイト数）

mic_MidB(Source as String, Start as Integer) as String

文字列の一部を返します(長さ指定）。（バイト数）

mic_LenMidB(Source as string, Start as Integer, Length as Integer) as String
