Micono ScreenCaptureについて

モニタの画像を得るためのpluginです。
REALbasic 3.x用のpluginです。

変更点：1.00b1->1.10b1
  • [Bug] (Win): Threadで使用すると落ちる不具合を修正
  • [Imp] (Car): まだ不安定

使い方

スクリーンキャプチャ画像を得る

mic_ScreenCapture( left as Integer, top as Integer, width as Integer, height as Integer ) as Picture
