Micono RBmovieUtil v1.5.0

プラグインは2001/7/8にアップロードしました。

[ダウンロード]

REALbasicのMovie Class, MoviePlayer Classのextension pluginです。
REALbasicで作ったQT関連のソフトで使用したプラグインです。
※REALbasicでは、Shared Libraryを直接利用する方法があります。その方法を使うのが面倒な場合に、お役立て下さい。

変更点

（ 1.2.1 b1 -> 1.5.0 b1）

  • [New] Max OS X対応
  • [New] (All) Movieのメソッドを４個追加
  • [New] (All) MovieTrack関連のメソッドを１４個追加
  • [New] (All) MediaHandler関連のメソッドを８個追加

（ 1.2.0 b1 -> 1.2.1 b1）

  • Fixed bug GetMovieCurrentPicture, GetMoviePicture of Windows.

（1.0.5 b2 -> 1.2.0 b1）

  • REALbasic 2.1.x 以降で起動可
  • プラグイン名を変更しました。以前使用していたMicono RBmovieUtil (R2)をPluginsフォルダから削除してください。
  • [New] Win32をサポート
  • [New] mic_REALenterMovies as Integer (Global method)
  • [New] mic_SaveAsMovie( m As Movie ) as Integer (FolderItem extension method)
  • [New] mic_UpdateMovieResource( m As Movie ) as Integer (FolderItem extension method)
  • [New] mic_FlattenMovieData( m As Movie ) as Integer (FolderItem extension method)

使い方

注）mp as MoviePlayer / m as Movie
あまり詳しく書いていませんので、サンプルと照らし合わせながらご使用下さい。

Global Methods

QuickTimeを使用可能にする（使用可能かを調べる）

result = mic_REALenterMovies

result = 0 ならば使用不可

FolderItem extension class

ムービーをアップデートする

err = f.mic_UpdateMovieResource( mp.movie )

err = 0でなければエラー

ムービーを保存する

err = f.mic_SaveAsMovie( mp.movie )

err = 0でなければエラー

ムービーを保存する

err = f.mic_FlattenMovieData( mp.movie )

err = 0でなければエラー
