MPCP Networking for REALbasic

MPCP Socket 1.0 is a bundle of classes that allow you to easily add sophisticated networking support to your REALbasic applications. The programmer's interface is similar to Apple Events but much simpler. The package currently is free of charge, so download it now and try it if you like.

Registry

New classes and commands should be registered. Simply make a list of all classes, their commands and the commands' optional and required arguments and send the list to me. This way, all applications using MPCP can be kept fully compatible to each other. Send the list to mpcp-registry@peppermind.comand I will report back to you.
 

Registered Suites

The registered commands are grouped into "suites" that each stand for a certain task or domain of application. The Standard Suite document describes the obligatory Core Suite, the File Suite for transfering files and the Chat Suite for presence information, buddy lists and text transfer. By now, none of them is complete in any way. If you have good ideas for commands, requirements and the like, please let me know!
