EasyPOP3 ReadMe:

EasyPOP3
By Navdeep Bains

What Is It?

EasyPOP3 is a POP3 socket. POP3 is the most common protocol used to access email.

Methods - Commands

LogOn
Variables: Address(string), User(string), Pass(string), Port(integer), UseAPOP(boolean)
Events: LoggedIn

This method tells EasyPOP3 to logon to a server using the specified user & pass.

GetCount
Variables: None
Events: CountReceived

Retrieves the number of waiting messages.

GetSizes
Variables: None
Events: SizesReceived

Retrieves the size of every message.

GetSize
Variables: Message Number(integer)
Events: None.

This method returns the size of the specified message(if you've called GetSizes).

GetUIDL
Variables: None
Events: GotUIDL

Retrieves a list of tags that identify each message throughout sessions.

GetHeaders
Variables: Message Number(integer)
Events: HeadersReceived

Retrieves the headers of the specified message.

GetAllHeaders
Variables: None
Events: HeadersReceived(called as the headers come in)

Retrieves the headers of all messages.

GetMessage
Variables: Message Number(integer)
Events: MessageReceived

Retrieves the body of the specified message.

GetAllMessages
Variables: None
Events: MessageReceived(called as the messages come in)

Retrieves the bodies of all messages.

DeleteMessage
Variables: Message Number(integer)
Events: DeleteSuccess

Deletes the specified message.

DeleteAll
Variables: None
Events: DeleteSuccess(called as the messages are deleted)

Deletes all messages, this method only works if you've called GetCount, GetUIDL, or GetSizes.

Reset
Variables: None
Events: ResetSuccess

Rescues all deleted messages.

Quit
Variables: None
Events: None

Closes the connection.

Methods - Other

SetNoopSecs
Variables: Seconds(integer)

Tells EasyPOP3 how long to wait before sending a NOOP command(to keep the connection alive)

Properties

Count

The message count, EasyPOP3 will set this property when you call GetCount, GetSizes, or GetUIDL.

BufferSize

Tells EasyPOP3 how large a message must be before it caches it to disk.
