RBfontUtilについて

フォントに関するREALbasic 2用のpluginです。
  • フォントＩＤからフォント名を得る。
  • フォント名からフォントＩＤを得る。
  • 指定したサイズのフォントがあるか調べる。
  • フォントの幅／高さなどの情報を得る。

　変更：

  • mic_GetFontInfoメソッド名を修正（b２）

使い方

フォントＩＤからフォント名を得る。

mic_GetFontName(fontNum as integer) as string

　

フォント名からフォントＩＤを得る。

mic_GetFontNumber(fontName as string) as integer

指定したサイズのフォントがあるか調べる。

mic_RealFont(fontNum as integer, fontSize as integer) as boolean

フォントの幅／高さなどの情報を得る。

mic_GetFontInfo( infoType as integer ) as integer

infoType：
０　Height above base line
１　Depth below base line
２　Width of widest character in the font
３　Distance between lines

例：

dim g as Graphics
dim ascent as integer

ascent = g.mic_GetFontInfo(0)
